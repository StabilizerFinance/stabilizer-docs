# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email:

**info@stabilizer.finance**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Disclosure Policy

- We will acknowledge receipt within 48 hours
- We will provide an initial assessment within 7 days
- We will work with you to understand and fix the issue
- We will credit you in our security acknowledgments (if desired)

## Bug Bounty

A bug bounty program will launch alongside our testnet in Phase 2.

## Security Audits

Contracts will undergo professional security audits before mainnet launch.

Audit reports will be published here upon completion.
